## Tiering LinkedIn Sales Navigator

### Tier 1 — Contacter en priorité : Coach Cabinet
- Associé ou coach certifié dans un cabinet structuré de coaching
  (Visconti Partners, Issho, Mozaïk, Linkup, Praesta, ou équivalent)
- Focus explicite : dirigeants, comités de direction, PME/ETI, transformation, performance
- Ancienneté dans le rôle : 3 ans+
- Background : ex-dirigeant, ex-DRH, ex-consultant senior

### Tier 2 — Contacter en second : Coach Solo
- Indépendant (micro-entreprise, société à son nom, self-employed)
- Focus dirigeants/PME clair dans la description
- Ancienneté : 2 ans+
- OU : coach cabinet avec infos insuffisantes pour confirmer Tier 1
- OU : coach solo avec description absente mais titre et parcours cohérents

### Tier 3 — Ne pas contacter
- Salarié dans une entreprise non-coaching (Stellantis, KEDGE, Leasys...)
- Coach académique ou institutionnel (programmes européens, publics)
- Focus uniquement particuliers / bien-être / sport / mindset
- Hors France
- Données insuffisantes après enrichissement

### Règle profils sans description

| Situation | Décision |
|-----------|----------|
| Cabinet identifiable + titre clair | Tier 1 |
| Titre clair + société à son nom + exp cohérente | Tier 2 |
| Titre vague + rien d'autre | Tier 3 |
